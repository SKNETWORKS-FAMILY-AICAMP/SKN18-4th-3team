# SKN18-4th-3team

## 프로젝트 구조

```
.
├── apps/                     # Django 애플리케이션들
│   ├── chatbot/              # 챗봇 기능 앱
│   ├── emotionreports/       # 감정 리포트 앱
│   └── users/                # 사용자 관리 앱
│
│
├── config/                   # Django 프로젝트 설정
│   ├── settings.py           
│   ├── urls.py               
│   ├── wsgi.py               
│   └── asgi.py
│
│               
├── pipeline/                 # 데이터 처리 파이프라인
│   ├── data_collect.py       # 데이터 수집
│   ├── embedding.py          # 임베딩 처리
│   └── preprocess_data.py    # 데이터 전처리
│
│
├── nodes/                    # 노드 관련 폴더
│   └── create_node.py        
│
│
├── services/                 # 공통 서비스
│   ├── llm_client.py         # LLM 클라이언트
│   ├── vector_store.py       # pgvector 검색
│   └── utils.py              # 유틸리티 함수
│
│
├── static/                   # 정적 파일
│   ├── css/                  # CSS 파일
│   └── js/                   # JavaScript 파일
│
│
├── templates/                # HTML 템플릿
│   ├── chatbot/              # 챗봇 템플릿
│   ├── emotionreports/       # 감정 리포트 템플릿
│   └── users/                # 사용자 템플릿
│
│
├── .env                       # 환경 변수
├── docker-compose.yml         # Docker 구성
├── manage.py                  # Django 관리 스크립트
└── requirements.txt           # Python 의존성
```

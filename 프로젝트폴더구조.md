### LangGraph 시스템 (graph/)
- **Nodes**: 작업 실행 (분류, 검색, 답변 생성 등)
- **Agents**: 분기 결정 (조건부 라우팅)
- **정보형 플로우**: 쿼리 분류 → 검색 → 답변
- **상담형 플로우**: 가이드라인 체크 → 증상 추출 → 검증 → 검색 → 답변

### RAG 시스템 (rag/)
- **Generation**: LLM을 통한 답변 생성
- **Retrieval**: 벡터 DB에서 관련 문서 검색
- **ETL**: 데이터 수집 → 변환 → 적재 파이프라인
